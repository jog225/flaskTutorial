# Python Flask Tutorial: Personal Blog

- To run on local host:
  - Windows: set FLASK_APP = flaskBlog.py
  - Mac: export FLASK_APP = flaskBlog.py
  - flask run

- To open debugger:
  - Windows: set FLASK_DEBUG = 1
  - Mac: export FLASK_DEBUG = 1
